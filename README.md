# 璃奈ちゃんbot TYPE2

このリポジトリは、同人誌「璃奈ちゃんを作ろう！TYPE2」の参考資料を公開するものです。

!["璃奈ちゃんを作ろう！TYPE2 表紙画像"](front_cover_small.png)

## ディレクトリ構成

```
etaoin_rinabot2
├── app : 主にPythonスクリプト
│   ├── board : 璃奈ちゃんboard
│   ├── drive : 璃奈ちゃんdrive
│   ├── etc : その他スクリプト
│   ├── frame : 璃奈ちゃんframe
│   ├── rinabot : 高レベルスクリプトやシナリオ実行スクリプト
│   ├── scenario : サンプルシナリオファイル
│   ├── slide : スライドショー機能
│   └── speech : 璃奈ちゃんspeech
├── elec : 各プリント基板のKiCadデータ
│   ├── board : 璃奈ちゃんboard制御基板
│   ├── hub : ハブ基板
│   ├── led_a : LEDのモジュール
│   └── motor_driver : 璃奈ちゃんdrive制御基板
└── fsw : ファームウェア
　   ├── board : 璃奈ちゃんboard
　   └── drive : 璃奈ちゃんdrive
```
